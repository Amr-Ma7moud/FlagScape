# FlagScape

An interactive web application for learning about world flags, capitals, and countries.

## Quick Start

1. **Start the backend server:**
   ```bash
   cd backend
   ./start.sh
   ```

2. **Open the application:**
   - Open `index.html` in your browser
   - Or serve via a local web server

## Project Structure

- **`/`** - HTML pages and main application files
- **`css/`** - Stylesheets
- **`js/`** - JavaScript game logic and utilities
- **`backend/`** - Python Flask API server
- **`assets/`** - Images and media files
- **`htmlUtil/`** - Reusable HTML components
- **`util/`** - Utility scripts
- **`docs/`** - All project documentation
  - `course/` - Course-specific documentation and requirements
  - `project/` - Technical and project documentation

## Games

- **Guess the Flag** - Identify countries by their flags
- **Speed Quiz** - Quick-fire flag identification
- **Capital Challenge** - Match capitals to countries
- **Flag Guesser** - Advanced flag guessing game
